# Notes From VoltAgent/awesome-agent-skills

`VoltAgent/awesome-agent-skills` is a curated index, not a directly installable skill. Use it as a discovery and quality reference.

Key lessons to apply:

- Prefer real-world skills from official teams or active community maintainers.
- Avoid bulk-generated or thin repositories with no evidence of use.
- Check platform paths because skill locations differ:
  - Codex project: `.agents/skills/`
  - Claude Code project: `.claude/skills/`
  - GitHub Copilot project: `.github/skills/`
  - Gemini CLI project: `.gemini/skills/`
  - Cursor project: `.cursor/skills/`
- Good skill metadata states what the skill does and when to use it.
- Good skill bodies use progressive disclosure: concise top-level instructions plus optional scripts/references.
- Do not treat inclusion in an awesome list as proof that a repository solves the user request. Inspect the linked repo and relevant skill files.

When building a shortlist, distinguish:

- Curated list entry
- Official skill
- Community skill
- Skill collection
- Related reference material
